## 📖 О проекте

Этот репозиторий содержит результаты молекулярно-динамического моделирования биомолекулярных систем. Основная цель — исследование структурной динамики и конформационной подвижности кофакторов (NADH, NADPH) в активных центрах ферментов.

### 🔬 Текущее исследование: ADH + NADH

Изучение **конформационного дребезга** NADH в алкогольдегидрогеназе (ADH) из *Drosophila melanogaster*. Под "дребезгом" понимаются низкоамплитудные колебания кофактора внутри активного центра, которые могут играть роль в каталитическом цикле.

**Ключевые вопросы:**
- Каковы амплитуды и частоты колебаний NADH?
- Какие водородные связи удерживают кофактор?
- Различается ли подвижность аденозиновой и никотинамидной частей?

### 📊 Система

| Параметр | Значение |
|----------|----------|
| **Белок** | Alcohol dehydrogenase (EC 1.1.1.1) |
| **Организм** | *Drosophila melanogaster* |
| **PDB ID** | [1MG5](https://www.rcsb.org/structure/1MG5) (1.6 Å) |
| **Кофактор** | NADH (NAI) |
| **Силовое поле** | CHARMM36-feb2026 + CGenFF 5.0 |
| **Вода** | TIP3P (~42 500 молекул) |
| **Температура** | 300 K |
| **Длительность** | 50 нс |

### 🛠 Методы

- **Параметризация:** единая топология белок + NADH + ацетат (`pdb2gmx -merge all`)
- **Релаксация:** минимизация → NVT (100 пс) → NPT (100 пс)
- **Продуктивный MD:** 50 нс, Parrinello-Rahman баростат, V-rescale термостат
- **Анализ:** RMSD, PCA, водородные связи, расстояния
